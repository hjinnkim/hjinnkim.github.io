Chirpy 테마로 새 Github blog를 개설하며 따랐던 링크를 정리한 게시글입니다.

## 개발환경

현재 Github blog는 Windows 11 + WSL2 / Ubuntu 20.04를 기반으로 작성됩니다.

## Ruby 설치

Jekyll 테마를 적용하기 위해 Ruby를 설치해야 합니다. WSL를 사용할 경우 사실상 Ubuntu와 동일하기 때문에 CLI로 Ruby를 설치하였습니다.

Ruby 설치 가이드는 다음 [링크](https://gorails.com/setup/ubuntu/20.04)를 쭉 따라가시면 됩니다. (Ruby / node.js 모두 설치하세요.)

## Github blog 개설

Github blog 개설 가이드는 구글에서 많이 찾을 수 있습니다. 그 중 다음 [블로그 포스트](https://www.irgroup.org/posts/jekyll-chirpy/)를 따라 [Chirpy 테마](https://github.com/cotes2020/jekyll-theme-chirpy)의 Github blog를 개설하였습니다. 

## 블로그 포스트 작성 가이드

Chirpy official website에서 영문버전의 포스트 작성 방법 가이드를 잘 정리해두었습니다. [(Link)](https://chirpy.cotes.page/posts/write-a-new-post/) 해당 가이드를 한국어로 번역해주신 블로그가 있습니다. 필요하다면 참고하면 될 것 같습니다. [(Link)](https://j1mmyson.github.io/posts/postingGuide/)

## 블로그 작성 목적

해당 블로그는 공부한 내용을 정리하는 것에 목적을 두기 때문에 커스터마이징에 그렇게 공을 들이지 않을 것 같습니다. 하지만 시간이 날 때, 커스터마이징 해볼 수 있도록 노력해보겠습니다.
